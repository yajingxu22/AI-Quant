# Technical Indicator Interactive Tool — 产品设计文档

## 1. 产品概述

一个纯前端 **HTML/JavaScript** 交互工具，用户可通过选择股票、调整技术指标参数，实时计算并绘制 RSI / MACD / 布林带 / ATR 四个经典指标，所有计算在浏览器端完成。

## 2. 用户故事

- 作为量化研究者，我想快速切换不同股票查看指标形态
- 作为交易员，我想调整 RSI 周期看不同参数下的信号差异
- 作为学习者，我想理解 MACD 的 fast/slow 参数变化对 DIF 线的影响
- 作为分析者，我想在同一张图上叠加多个指标综合判断

## 3. 功能需求

### 3.1 数据层

| 模块 | 功能 |
|:----|:----|
| 数据源 | 预置已下载的 CSV 数据（本地 data/raw/ 目录） |
| 股票切换 | 下拉选择已配置的股票（三一重工、比亚迪、长江电力、中芯国际） |
| 日期范围 | 可选：1年 / 6个月 / 3个月 / 自定义 |
| 数据缓存 | 首次加载后缓存在浏览器内存中，切换无需重新请求 |

### 3.2 指标计算（浏览器端 JavaScript 实现）

| 指标 | 可调参数 | 默认值 | 输出字段 |
|:----|:---------|:------|:---------|
| RSI | period | 14 | rsi |
| MACD | fast / slow / signal | 12 / 26 / 9 | dif, dea, hist |
| 布林带 | period / std_dev | 20 / 2 | upper, mid, lower, bandwidth, %b |
| ATR | period | 14 | tr, atr |

### 3.3 界面功能

| 模块 | 功能 |
|:----|:----|
| 控制面板 | 股票下拉选择 + 日期范围 + 四个指标各自的参数滑块/输入框 |
| 图表区域 | 价格 + 各指标面板（可切换/可缩放） |
| 信号汇总 | 当前最新信号状态（超买/超卖/金叉/死叉等） |
| 数据表格 | 展示最近 N 天的原始数据 + 指标值 |

### 3.4 交互细节

- 每个指标旁有 **启用/禁用** 开关，控制该指标是否显示
- 参数调整后 **立即重绘**，无需手动触发
- 图表支持 **鼠标悬浮查看数值**（tooltip）
- 滑块调整时实时显示当前数值
- "重置"按钮一键恢复所有默认参数

## 6. 组件说明

### 6.1 StockSelector (股票选择器)
- 预置股票列表，从 data/raw/ 读取
- 选择后加载对应 CSV

### 6.2 ParameterPanel (参数面板)
- 每个指标一个折叠面板
- 含参数滑块/数字输入框和启用开关

### 6.3 IndicatorCalculator (指标计算器)
- 纯 JS 实现四个指标算法
- 输入: OHLC 数组 + 参数对象
- 输出: 带时间戳的指标值数组

### 6.4 ChartRenderer (图表渲染器)
- 基于 ECharts
- 4 面板布局 (grid)
- 响应式 / tooltip / 数据缩放

### 6.5 SignalSummary (信号汇总)
- 展示最新值的文字解读
- MACD 金叉/死叉检测
- RSI 超买/超卖提示

## 8. 非功能需求

- **纯前端**：无后端依赖，所有计算在浏览器 JS 中完成
- **性能**：240 条日线数据下，参数调整到图表重绘 < 100ms
- **兼容性**：Chrome / Firefox / Safari 现代浏览器
- **数据格式**：CSV UTF-8 with BOM
- **技术栈**：原生 HTML + CSS + JS + ECharts CDN
